# 🍽️ Sistema de Gerenciamento de Restaurantes (Python)

Este projeto é um **sistema simples de gerenciamento de restaurantes**, desenvolvido em **Python**, utilizando conceitos fundamentais da linguagem como:

- Listas e dicionários  
- Funções  
- Estruturas condicionais  
- Laços de repetição  
- Tratamento de exceções (`try/except`)  
- Interação com o usuário via terminal  

O sistema funciona totalmente no **terminal/console** e permite cadastrar, listar e alterar o estado (ativo/desativado) de restaurantes.

---

## 📌 Funcionalidades

- 📋 Listar restaurantes cadastrados  
- ➕ Cadastrar novos restaurantes  
- 🔄 Ativar ou desativar restaurantes  
- ❌ Encerrar o aplicativo  

Cada restaurante é armazenado como um **dicionário**, contendo:
- `nome`
- `categoria`
- `ativo`

Todos os restaurantes são armazenados em uma **lista**.

---

## 🧠 Conceitos Aplicados

- Estrutura de dados com **listas e dicionários**
- Organização do código em **funções**
- Uso de **menu interativo**
- **Tratamento de erros** para entradas inválidas
- Manipulação do terminal com `os.system('cls')`
